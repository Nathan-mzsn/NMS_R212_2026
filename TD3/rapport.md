
--Utilité du JavaScript--

      Quelles sont les choses qu'un site Web interactif peut faire et qu'un site purement HTML/CSS 
      ne peut pas faire ?

Un site web avec du JavaScript peut par exemple intégrer un formulaire que l’utilisateur remplit et modifie une variable. Cette dernière peut ensuite produire différents effets en fonction de la réponse donnée dans le formulaire. Grâce au JavaScript on peut ajouter un commentaire sur les réseaux, afficher un pop-up ou encore permettre de remplir et d’envoyer un message de réclamation sur un site d’e commerce.
