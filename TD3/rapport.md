## Points clés

### 1. Expliquez le modèle événement → état → rendu avec un exemple de votre choix

- D'abord l'événement permet de décrire l'action de l'usager (clique sur la zone de formulaire).
Ensuite l'état décrit l'état d'un objet après l'action (zone de texte = ouverte).
Enfin le rendu décrit ce qui à changer (ajouter classe .formulaire-isopen).

### 2. Expliquez en quoi classList.toggle est la passerelle entre JS et CSS 

- Ici le Javascript ajoute les classes, le CSS leur donne des propriétés graphiques.

## Question propre au TD
### Quelles sont les choses qu'un site Web interactif peut faire et qu'un site purement HTML/CSS ne peut pas faire ?

- Un site web avec du JavaScript peut par exemple intégrer un formulaire que l’utilisateur remplit et modifie une variable. Cette dernière peut ensuite produire différents effets en fonction de la réponse donnée dans le       formulaire. Grâce au JavaScript on peut ajouter un commentaire sur les réseaux, afficher un pop-up ou encore permettre de remplir et d’envoyer un message de réclamation sur un site d’e commerce.
